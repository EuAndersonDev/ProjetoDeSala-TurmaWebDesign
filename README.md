# Projeto de Sala (Exemplo)

> Pequeno projeto de exemplo criado durante as aulas de Web Design.

## Descrição

Este diretório contém uma página estática de exemplo usada nas atividades da sala. É um projeto simples para praticar HTML e CSS.

## Estrutura

- `index.html` : página principal do projeto.
- `style.css` : estilos aplicados à página.
- `assets/` : imagens e outros recursos usados pelo projeto.

## Como usar

1. Abra o arquivo `index.html` no navegador (duplo-clique ou arraste para a janela do navegador).
2. Para testes de desenvolvimento, recomenda-se usar um servidor local (ex.: extensão Live Server do VS Code).

## Notas

- Arquivos criados para demonstração e exercícios de aula.
- Sinta-se à vontade para modificar os arquivos e testar diferentes estilos.
